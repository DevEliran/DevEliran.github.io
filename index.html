Image Compression

 Code
Using SVD decomposition, I created a script that takes an image and shows it compression in a gradual way.
I wrote an article on Medium that was published on Cantor's Paradise (Medium biggest math publication) covering the math behind the compression.
Just as a side note, the original picture was around 3000 x 2000 and this script compressed it to 50 x 50 and details are quite clear.
Article

Parallel Game Of Life

 Code
Implementing Game Of Life by John Conway with a twist - there are 8 species instead of one and there are two phases for each generation. I implemented this in a multi-threaded approach while implementing on my own a semaphore and a producer-consumer queue in c++.

League of Legends - Win Prediction

 Code
Using a dataset on the video game League of Legends I created an estimator that predicts the winner of a game based on the stats of it.
To predict the winner, first I had to determine which parameters of the game has a good correlation with winning and then training the dataset based on them.
This model was able to predict a win accurately in 90.3%.
This project was implemented with tensorflow 2.0.

Bash Clone - Linux Command Line

 Code
Implementing a new linux command line - ’smash’.
This command line natively supports commands such as ls, cd, showpid, chprompt, pwd, jobs, quit, kill, fg, bg including pipes and IO redirection.
All other commands are supported through bash.
